## Description

[AICON](https://github.com/curiosityprocessor/bits-n-pieces) sub-repository containing AI generated Contents.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
